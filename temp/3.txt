## 3Ô∏è‚É£ **Notification Toast (Success/Error)**

```jsx
export function Toast({ type = "success", message }) {
  const colors = {
    success: "bg-success text-white",
    error: "bg-error text-white",
    info: "bg-info text-white"
  };
  return (
    <div className={`fixed top-4 right-4 p-4 rounded-lg shadow-lg ${colors[type]} animate-fadeIn`}>
      {message}
    </div>
  );
}
```

> ‚ö° Add `animate-fadeIn` using `@tailwindcss/animations` or Framer Motion if needed.
